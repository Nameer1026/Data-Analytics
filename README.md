# HR Attrition Analysis Dashboard

**Tools:** Power BI · DAX

An interactive dashboard analyzing 1,400+ employee records to identify the strongest drivers of attrition — giving HR a data-backed starting point for retention planning instead of relying on anecdotes.

## Business Problem
HR suspected attrition was rising in specific teams but had no structured way to test *why* — was it overtime? job satisfaction? demographics? This dashboard isolates the actual drivers.

## What It Does
- Tracks **attrition rate** broken down by department, demographics, overtime status, and job satisfaction
- Surfaces which factors correlate most strongly with employees leaving
- Interactive views let HR test hypotheses (e.g., "is overtime really the driver, or is it tenure?")
